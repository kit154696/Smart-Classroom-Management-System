# โปรเจกต์ Smart Classroom Management System

ระบบนี้เป็นระบบจัดการห้องเรียนอัจฉริยะแบบครบวงจรเลย (มีทั้งเช็คชื่อ, จัดการคะแนน, ดูตารางเรียน, แล้วก็วิเคราะห์ความเสี่ยงนักศึกษา)

ตัวโปรเจกต์เราแบ่งการทำงานเป็น 2 ส่วนนะ:
* **หน้าบ้าน (Frontend):** ใช้ HTML / CSS / JavaScript ปกติเลย
* **หลังบ้าน (Backend):** ใช้ Node.js + Express แล้วต่อกับ Database เป็น MySQL

---

## เครื่องมือที่ต้องมีในเครื่อง
ก่อนจะเริ่มรัน เช็คก่อนนะว่าในเครื่องมีโปรแกรมพวกนี้หรือยัง:
1. **Node.js:** แนะนำให้โหลดเวอร์ชัน LTS มาลงนะ 👉 [ดาวน์โหลดที่นี่](https://nodejs.org/)
2. **MySQL / TablePlus:** เอาไว้เปิดดูและจัดการฐานข้อมูล
3. **VS Code:** หรือใครถนัด Editor ตัวอื่นก็ใช้ได้เลย

---

## วิธีตั้งค่าและรันโปรเจกต์ในเครื่องตัวเอง

### 1. โหลด Library ที่จำเป็น
เปิด Terminal ขึ้นมา แทร็กมาที่โฟลเดอร์โปรเจกต์นี้ แล้วพิมพ์คำสั่งนี้เพื่อให้มันโหลดของที่ต้องใช้:
```bash
npm install
2. สร้างฐานข้อมูล (Database)
เปิดโปรแกรมจัดการ Database (เช่น TablePlus)

ก๊อปปี้โค้ดทั้งหมดในไฟล์ index.sql ไปวางแล้วกด Run ได้เลย มันจะสร้างฐานข้อมูลชื่อ smart_classroom พร้อมสร้างตารางและใส่ข้อมูลจำลอง (Mock Data) ให้เสร็จสรรพ

3. เชื่อม Database ในไฟล์ server.js (⚠️ สำคัญมาก)
ตอนนี้ในไฟล์ server.js เราเขียนตั้งค่าไว้แบบใช้ process.env เพื่อเตรียมเอาขึ้นโฮสต์จริง
ถ้าเพื่อนๆ จะรันเทสในเครื่องตัวเอง (Localhost) อย่าลืมไปแก้โค้ดช่วงบนๆ ใน server.js ให้เป็นข้อมูลเครื่องตัวเองชั่วคราวก่อนนะ หน้าตาประมาณนี้:

JavaScript
const db = mysql.createConnection({
  host:     'localhost',
  user:     'root',
  password: 'ใส่รหัสผ่านฐานข้อมูลของเครื่องตัวเองตรงนี้นะ', 
  database: 'smart_classroom',
  port:     3306
});
4. สตาร์ท Backend เซิร์ฟเวอร์
เปิด Terminal แล้วรันคำสั่ง:

Bash
npm start
(หรือจะใช้ node server.js ก็ได้เหมือนกัน)
ถ้าทำถูกหมด มันจะขึ้นข้อความว่า: ✅ เชื่อมต่อฐานข้อมูล MySQL สำเร็จแล้ว!

5. เปิดดูหน้าเว็บ
พอเซิร์ฟเวอร์รันแล้ว เข้าเบราว์เซอร์ไปที่ลิงก์นี้ได้เลย: 👉 http://localhost:3000

📂 ไฟล์ไหนใช้ทำอะไรบ้าง?
index.html : หน้าตาเว็บทั้งหมดรวมอยู่ที่นี่ (เขียน UI และใช้ JavaScript ดึง API ในไฟล์เดียวเลย)

server.js  : ตัวหลังบ้าน (API) เอาไว้จัดการ Routing และรับส่งข้อมูลกับ MySQL

index.sql  : โค้ดสร้างโครงสร้าง Database และข้อมูลจำลอง

package.json : รายชื่อแพ็กเกจที่โปรเจกต์นี้ใช้ (พวก Express, MySQL2, Cors)
